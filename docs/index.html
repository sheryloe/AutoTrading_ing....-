Model A: Trend Following (지수 추종)

Low Risk

장기 이동평균선을 활용하여 시장의 주류 추세에 올라타는 전략입니다. 노이즈를 제거하기 위해 RSI 필터를 결합합니다.

# Entry Logic
Signal: \(EMA_{20} > EMA_{60} \text{ and } EMA_{60} > EMA_{120}\)
Filter: \(RSI(14) < 65 \text{ (과매수 방지)}\)

Model B: Mean Reversion (평균 회귀)

Mid Risk

가격이 통계적 변동 범위를 벗어났을 때 다시 평균으로 돌아오려는 성질을 이용한 역추세 전략입니다.

# Entry Logic
Signal: \(Price < LowerBB(20, 2)\)
Volume: 전일 평균 거래량 대비 150% 이상 폭증 시 진입

Model C: Volatility Breakout (변동성 돌파)

High Reward

전일 가격 변동폭에 특정 계수(K)를 곱한 값을 돌파할 때 강한 매수세가 붙는 원리를 이용합니다.

# Entry Logic
Target: \(Open + (PrevHigh - PrevLow) \times 0.5\)
Exit: 당일 종가 일괄 매도 (Risk Hedge)

Model D: Deep Learning (LSTM 예측)

AI Based

3층 LSTM 레이어를 통해 과거 60일간의 OHLCV 데이터를 시계열 분석하여 향후 4시간의 방향성을 예측합니다.

# Model Spec
Architecture: \(LSTM(64) \to Dropout(0.2) \to Dense(1)\)
Optimizer: Adam (Learning Rate: 0.001)
