System Overview

본 대시보드는 4가지 독립 전략을 병렬 운용하는 실시간 퀀트 매매 시스템입니다.

MODEL A: Trend Follower

EMA 정배열 구간에서 RSI 지표를 필터링하여 강력한 추세 상승 시에만 진입합니다.

MODEL B: Mean Reversion

볼린저 밴드 하단을 이탈하는 과매도 시점에서 단기 기술적 반등을 노립니다.

MODEL C: Volatility Breakout

전일 변동폭 대비 특정 수치(K)를 돌파할 때 모멘텀에 올라탑니다.

MODEL D: Deep Learning

LSTM 신경망으로 과거 60일의 데이터를 학습해 방향성을 예측합니다.
