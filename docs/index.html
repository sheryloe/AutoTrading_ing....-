Model A: Trend Follower

STRATEGY B

Model B: Mean Reversion

가격이 평균에서 과도하게 멀어질 때 발생하는 회귀 본능을 이용합니다. 볼린저 밴드의 하단 터치와 거래량 급감을 분석하여 단기 반등 포인트를 포착하며, 15분봉 기준 스캘핑에 최적화되어 있습니다.

Bollinger-Bands
Mean-Recap
Scalping
STRATEGY C

Model C: Volatility Breakout
