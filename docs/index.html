System Architecture

Automethemoney는 Python 기반의 매매 봇과 GitHub Actions의 스케줄러를 결합한 완전 자동화 시스템입니다. 시장의 노이즈를 제거하기 위해 4가지 독립된 전략(Multi-Strategy)을 병렬 가동하며, 각 모델의 리스크 한도를 동적으로 관리합니다.

How it works:

  1. 1. GitHub Actions 트리거 (매일 정해진 시간)
  2. 2. Python 매매 엔진 구동 (API 연결)
  3. 3. 모델별 데이터 분석 및 주문 체결
  4. 4. 결과 JSON 저장 및 대시보드 업데이트

Technical Models Logic

Model A (Trend Following)

이동평균선 정배열 구간에서 RSI 필터를 통과한 강세 종목만 추종합니다. \(EMA_{20} > EMA_{60}\) 조건 충족 시 진입.

Model B (Mean Reversion)

볼린저 밴드 하단(\(2\sigma\)) 이탈 시 단기 반등을 노립니다. 거래량 감소가 확인된 지점에서 매수 진입.
