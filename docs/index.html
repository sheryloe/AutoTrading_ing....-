EMA 골든크로스 & RSI 필터

지수이동평균(EMA)의 정배열 상태에서 가격이 20일 이평선에 눌림목을 형성할 때 진입합니다. 과매수 구간에서의 추격 매수를 방지하기 위해 RSI 지표를 결합합니다.

  • Signal: \(EMA_{20} > EMA_{60} > EMA_{120}\)
  • Entry: \(Price \leq EMA_{20} \times 1.01\) AND \(RSI_{14} < 65\)
  • Exit: \(EMA_{10}\) 이탈 또는 고점 대비 3% 익절/손절
Win Rate: 58% Risk/Reward: 1:2.5

  • Filter: \(Volume < AvgVolume_{20} \times 0.7\)
  • Target: \(BB_{Middle}\) (20일 평균선) 터치 시 청산

K-Value 변동성 돌파

전일 고가와 저가의 변동폭에 특정 계수(\(K\))를 곱한 값을 당일 시가에 더해 돌파 시 매수합니다. 장 종료 시 전량 매도하여 오버나이트 리스크를 배제합니다.

  • Formula: \(Target = Open + (PrevHigh - PrevLow) \times K\)
  • Parameter: \(K = 0.5\) (최적화 가변값)
  • Exit: 당일 종가(Market Close) 일괄 청산
