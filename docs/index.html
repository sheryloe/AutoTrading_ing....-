Technical Model Specs

Model A: Trend Follower

주요 로직: 이동평균선(EMA 20/60/120)의 정배열 상태에서 발생하는 강력한 추세에 올라타는 전략입니다.

진입 필터: RSI(14)가 65 이하이면서 가격이 EMA 20선에 눌림목을 형성할 때 진입하여 추격 매수를 방지합니다.

Model B: Mean Reversion

주요 로직: 가격이 통계적 변동 범위인 볼린저 밴드 하단(\(2\sigma\))을 이탈할 때 단기 반등을 노리는 회귀 전략입니다.

진입 필터: 거래량 분석(VSA)을 통해 하락 압력이 소멸되는 거래량 급감 시점을 포착하여 승률을 높입니다.

Model C: Volatility Breakout

주요 로직: 래리 윌리엄스의 변동성 돌파 전략을 기반으로 하며, 전일 변동폭(\(High - Low\))의 특정 비율(\(K\))을 돌파할 때 매수합니다.

진입 필터: 당일 장 마감 시 전량 매도하는 데이트레이딩 원칙을 준수하여 오버나이트 리스크를 제거합니다.

Model D: Deep Learning

주요 로직: LSTM(Long Short-Term Memory) 신경망을 활용하여 과거 시계열 데이터를 학습하고 향후 1시간 내 가격 방향성을 예측합니다.

진입 필터: 모델의 예측 확률(Softmax Output)이 75% 이상일 때만 가중치를 부여하여 진입 신호를 생성합니다.
