# AutoTrading_ing....- 노션 페이지 구조 (3-Step)

상위 페이지 제목은 **AutoTrading_ing....- | 자동매매 실험 콘솔 제작기** 로 두고, 아래 3개 하위 페이지를 넣으면 됩니다.

## 권장 설명

데모 자동매매 실험을 운영 콘솔, 런타임 설정, 리포트 관점으로 재정비한 과정을 3단계 노션 글로 정리한 세트입니다.

## 권장 페이지 트리

- Step 1. 자동매매를 기능보다 운영 리스크로 다시 설계한 이유
- Step 2. Flask 대시보드와 런타임 제어 레이어 구현 과정
- Step 3. 초기화 안전장치와 리포트 구조로 운영성을 마무리한 결과
